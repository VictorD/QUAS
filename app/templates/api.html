Questions

Resource Usage
GET /questions/

If no parameters given returns all the questions currently in the database. Otherwise there are options for pagination, parameters are wrapped in a paginate key value i.e {'paginate': parameters}.

Pagination Parameters

page_size
optional 		page size
page
optional 		the page number
order_by
optional 		order results by date/vote/name
filter_by
optional 		filter results by author/tags
filter_data
optional 		which author or tags to filter by
POST /questions/

Require user access

Posts a question to the database.

Parameters

title
required 		title for the question
body
required 		the question body
tags
optional 		a list of tags, tags which don't exist are added to database
PUT /questions/{id}/

Requires author

Updates a question.

Parameters

title
optional 		title for the question
body
optional 		the question body
DELETE /questions/{id}/

Requires author

Deletes a question.

GET /questions/{id}/replies

Gets replies for a the question.

GET /questions/{tagname}

Returns all questions that are tagged with the wanted tag name.

Replies

Resource Usage
GET /replies/

Returns all replies to all questions.

GET /replies/{id}

Returns a reply for a reply id.

POST /replies/

Require user access

Posts a reply to a question.

Parameters

body
required 		body for the reply
question_id
required 		the target question which to reply to
PUT /replies/{id}/

Requires author

Updates a reply.

Parameters

body
optional 		the reply body
DELETE /replies/{id}/

Requires author

Deletes a reply.

Tags

Resource Usage
GET /tags/

Get all tags.

Users

Resource Usage
GET /u/

Get all users.

GET /u/{id}/

Get a single user by id.

PUT /u/{id}/

Requires user

Updates a user.

Parameters

username
optional 		the user username
description
optional 		the user description
GET /u/login/

Redirects user to Google OpenID to let server access the users email, if user is not registered it will be when OpenID verified.

GET /u/logout/

Releases information on the user and clears QUAS related cookies.

value

value
